# biomedicalVis_project
You can find the visualisation via this link: https://biomedicalvisproject.streamlit.app/  
or run it locally in your terminal using streamlit run app.py

For this to run successfully make sure that gpd_worldwide.csv is included in your IDE.
The test dataset is data_water from https://www.who.int/data/gho/data/themes/topics/sdg-target-6-ensure-availability-and-sustainable-management-of-water-and-sanitation-for-all
